# Ultrasonic-sensor
This project measures the distance of objects using  an ultrasonic sensor and Arduino UNO and displays the result in real time on serial monitor or LED.
## Components Used
- HC-SR04 Ultrasonic Sensor
- Arduino Uno
- LED
- Jumper wires
- Buzzer
